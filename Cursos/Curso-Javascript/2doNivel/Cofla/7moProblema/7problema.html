7mo problema del curso de javascript

Septimo parte de la historia de Cofla

Problema 1

Cofla se quiere comprar una computadora Full HD

Datos

  • Debe ser full HD

Solucion

  • Hacer un sistema que le diga que computadora es full HD a partir de sus medidas
  • Y que le pregunte si esta seguro de comprar esa

Problema 2

Cofla entro a un sitio web y no sabe cual es

Datos

  • Cofla quiere saber el protocolo, Url, etc. de la pagina web

Solucion

  • Hacer un sistema que le muestre a Cofla en que sitio web entro
